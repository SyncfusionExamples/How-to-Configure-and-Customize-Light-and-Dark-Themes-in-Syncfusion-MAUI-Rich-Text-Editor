# Theme Customization in Syncfusion Rich Text Editor (.NET MAUI)

This sample demonstrates how to customize the **Light** and **Dark** themes in the Syncfusion [SfRichTextEditor](https://www.syncfusion.com/maui-controls/maui-rich-text-editor) for .NET MAUI.  
It also shows how to modify the toolbar appearance (text color, background color, and selection highlight color) using color pickers.

## What this sample includes

- Switching between **Material Light** and **Material Dark** themes  
- Updating toolbar text color  
- Updating toolbar background color  
- Customizing selection highlight color  
- Applying theme changes dynamically at runtime  
- Built in toolbar with commonly used formatting options

## How it works

- A toggle switch is used to change between light and dark themes.  
- Color pickers allow users to modify toolbar and selection highlight colors.  
- Theme resources are retrieved from the application merged dictionaries and updated at runtime.

## Requirements

- .NET MAUI  
- Syncfusion MAUI RichTextEditor  
- Syncfusion MAUI Inputs  
- Syncfusion MAUI Themes  

## Output

The sample demonstrates a Rich Text Editor where users can switch between themes and customize UI colors interactively.

# How-to-Configure-and-Customize-Light-and-Dark-Themes-in-Syncfusion-MAUI-Rich-Text-Editor
Learn how to configure and customize Light and Dark themes in the Syncfusion .NET MAUI Rich Text Editor. This guide explains theme switching, color customization, system theme detection, and styling options to help you deliver a consistent rich-text editing experience across all MAUI platforms.
